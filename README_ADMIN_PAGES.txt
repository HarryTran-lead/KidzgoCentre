
KidzGo Admin Pages
==================
Các trang quản trị cho my-center-fe (Next.js App Router + Tailwind).

Thêm vào dự án:
1) Giải nén nguyên thư mục vào gốc `my-center-fe` (nó chỉ **thêm file**).
2) Cài icon: `npm i lucide-react`
3) Chạy: `npm run dev`
4) Truy cập: /admin, /admin/students, /admin/teachers, /admin/classes, /admin/courses, /admin/rooms, /admin/fees, /admin/reports
